# Changelog

## 1.0.1

- Initial VS Code port of the DevAI IntelliJ plugin (`com.bmo.devai.intellij`).
- Dual-mode execution: GitHub Copilot CLI (SDK) and the VS Code Language Model API (Chat), with auto fallback.
- Test generation (single + folder), test scenarios & cases, shakedown suites.
- Documentation: inline, file, folder, README, doc-update, business summary, user stories.
- Code review (selection / changed files), API drift detection, findings tree + diagnostics.
- Diagrams: class / sequence / flow (Mermaid), render to SVG/Draw.io, update diagram.
- Feature code generation / update / scaffold; dependency analysis & migration; platform upgrades.
- SAST remediation via SonarQube and GitHub Code Scanning; Jira ticket fetch.
- RAG prompt enrichment, analytics/audit reporting, input filtering — ported verbatim from the plugin's resources.
